# Barbell Calculator

A simple barbell plate calculator that helps you figure out which plates to load on each side of the bar to reach your target weight.

## Features

- Enter your target weight and see which plates to load per side
- Visual barbell representation showing plate distribution
- Configurable plate inventory (set how many of each plate you own)
- Support for fractional plates (1lb, 0.75lb, 0.5lb, 0.25lb)
- Remembers your settings and last weight via local storage
- Shows actual weight achieved and any remainder

## Usage

1. Enter your target weight in pounds
2. The calculator shows which plates to load on each side
3. Click "Settings" to configure your available plate inventory and bar weight
